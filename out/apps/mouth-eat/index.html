MouthEat

口を開けて食べまくれ！リアルタイム食べゲーム

Open wide and eat! A real-time mouth-controlled eating game

Game Overview

口の動きをリアルタイム検出して食べ物をキャッチする新感覚ゲーム。画面を流れてくる100種類以上の食べ物を、実際に口を開けてパクパク食べよう！食べられないものには気をつけて。

Game Modes

⏱️

タイムアタック

  • •制限時間60秒
  • •1レーン
  • •時間とともにスピードアップ
❤️

サバイバル

  • •HP制（初期HP: 100）
  • •3レーン（高難易度）
  • •食べられない物でダメージ
🔥

コンボチャレンジ

  • •コンボ継続がカギ
  • •なし → Warming → Hot → Blazing → Legendary
  • •コンボが途切れたらゲームオーバー
🧠

ジャッジメント

  • •ベルトコンベア方式
  • •全35アイテム（75%食用, 25%非食用）
  • •正しく判断して食べよう
